üîß Core Components of n8n

1 Node

What it is: The building block of n8n. Each node performs ONE specific task.

Think of it as: A single instruction or action in your workflow.

What it is: The starting point that begins your workflow when something happens.

3 Action Node

What it is: Performs an action AFTER the trigger fires. This is where the work gets done.

üí¨ Send Slack message üìä Save to Google Sheets ‚úâÔ∏è Send email via Gmail üìû Create task in Asana

4 Flow/Workflow

What it is: The complete chain that connects trigger + actions together.

5 Variables

What it is: Temporary storage for data as it moves through your workflow.

Use for: Storing email content, form responses, API data, calculated values.

Data passes from node to node, getting processed along the way.
